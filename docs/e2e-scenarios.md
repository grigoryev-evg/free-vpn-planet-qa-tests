# E2E Scenarios — Пошаговые инструкции для ручного воспроизведения

> **Назначение:** документ для ручного тестирования автоматизированных E2E сценариев.
> Каждый сценарий содержит таблицу: номер шага → действие → ожидаемое поведение → фактическое (заполняется тестировщиком).
> Основано на `tests-e2e/tests/specs/assignment.e2e.ts` и `tests-e2e/features/assignment.feature`.

---

## Общие предусловия

- Браузер в режиме инкогнито / чистый профиль (без cookies, localStorage).
- Сайты доступны по HTTPS.
- Стабильное интернет-соединение.

## 1. TC_ASSIGN_001 — Sign Up (RU) + monthly + card → hosted checkout

**Теги:** `@assignment` `@smoke`  
**Стартовая страница:** https://freevpnplanet.com

| № | Действие | Ожидаемое поведение | Фактическое (заполнить) |
|---|----------|---------------------|-------------------------|
| 1 | Открыть https://freevpnplanet.com | Загрузилась главная страница freevpnplanet.com, видна кнопка «Sign Up» / «Регистрация» | |
| 2 | Перейти на страницу регистрации (кнопка «Sign Up») | Открылась форма регистрации с полями email, password | |
| 3 | Ввести уникальный email (напр. `test+20260517@example.com`) | Email принят, нет ошибки валидации | |
| 4 | Ввести валидный пароль (мин. 8 символов, цифры, заглавная) | Пароль принят, нет ошибки валидации | |
| 5 | Нажать «Next» / «Continue» | Открылся шаг выбора способа оплаты (видны варианты: bank card, crypto, etc.) | |
| 6 | Выбрать способ оплаты «Bank Card» | Карточная опция подсвечена/выбрана | |
| 7 | Отметить чекбокс принятия условий | Чекбокс отмечен | |
| 8 | Нажать «Pay» / «Submit» | Произошёл редирект на hosted-страницу платёжного провайдера (stripe.com, cloudpayments.ru или аналог) | |
| 9 | Проверить URL страницы | Домен не равен `freevpnplanet.com` — это разрешённый платёжный провайдер | |

## Примечания

- **Email:** Использовать уникальный email при каждом прогоне (добавлять timestamp: `test+YYYYMMDDHHmm@example.com`).
- **Пароль для Sign Up:** Берётся из `.env` переменной `SIGNUP_PASSWORD` (мин. требования: 8+ символов, цифры, заглавная буква).
- **Платёжные провайдеры:** Список разрешённых хостов — в `tests-e2e/tests/browser/pages/payment_page.page.ts` (метод `expectAllowedProviderHost`).
- **Автоматизированные тесты:** `npx playwright test --grep @assignment --project=chromium` (7 тестов, ~40 секунд).
- **Feature-файлы:** `tests-e2e/features/assignment.feature` (Gherkin-спецификация).
